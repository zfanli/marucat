# 从需求来看 MongoDB

## Introduction

从需求来看 MongoDB，了解能满足我们需求的最少量用法。

面向 MongoDB 的初学者。第一次阅读建议阅读全文，之后建议检索关键字或者根据索引按需阅读。

这一次我们使用 MongoDB 来储存文档，制作一个博客网站。

我们要了解文章、评论和设定，诸如此类的数据，如何储存在 MongoDB，以及如何使用和更新它们。

本文讨论的内容：

- 设计符合需求的数据结构
- document 的增删改查
- 计划获取的字段或内容
- 查询和过滤内嵌数组

基本上，就是入门知识。

## 最简内容索引

- find 查询
- project 计划字段
- aggregate 聚合
- update 更新
- delete 删除
